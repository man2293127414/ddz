# 局域网斗地主

#### 介绍
摸鱼神器，控制台斗地主！
最简单的局域网斗地主，只需要少量代码，就能实现在任何封闭的网络环境，来局美美的斗地主！


#### 软件架构
1. java1.8
2. maven
3. hutool
4. fastjson

#### 使用说明

1.  先启动GameServer()；
2.  根据ip端口启动GameClient()
